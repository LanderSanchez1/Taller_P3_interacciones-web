# Evaluador Académico en JavaScript

## Descripción

Este proyecto corresponde al desarrollo de un **Evaluador Académico** utilizando **HTML, Bootstrap y JavaScript**. La aplicación permite ingresar la información de un estudiante, calcular su promedio, determinar su estado académico y mostrar diferentes estadísticas relacionadas con su rendimiento. Además, incorpora un historial de estudiantes evaluados y diversas funcionalidades para facilitar la gestión de la información.

## Funcionalidades

- Registro del nombre y carrera del estudiante.
- Ingreso de cuatro notas con valores decimales.
- Cálculo del promedio general.
- Cálculo del promedio especial sin la nota más baja.
- Identificación de la nota mayor y la nota menor.
- Clasificación del estado académico (Aprobado, Recuperación o Reprobado).
- Clasificación del rendimiento (Alto, Medio, Básico o Bajo).
- Generación de recomendaciones académicas.
- Asignación automática de becas según la carrera y el promedio.
- Visualización de los resultados en formato JSON.
- Historial de estudiantes evaluados.
- Validación de nombres repetidos.
- Búsqueda de estudiantes por nombre.
- Listado de estudiantes en una tabla.
- Estadísticas generales del curso.
- Ranking de estudiantes según su promedio.

## Tecnologías utilizadas

- HTML5
- Bootstrap 5
- JavaScript (ES6)

## Cómo ejecutar la aplicación

1. Descargar o clonar el proyecto.
2. Abrir la carpeta del proyecto.
3. Ejecutar el archivo **index.html** en cualquier navegador web moderno (Google Chrome, Microsoft Edge o Mozilla Firefox).
4. Ingresar los datos del estudiante y utilizar los botones disponibles para evaluar, buscar o listar los registros.
